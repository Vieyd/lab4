Этот проект реализует алгоритм шифрования \doxylink{class_r_c4}{RC4} на языке Java. Алгоритм \doxylink{class_r_c4}{RC4} представляет собой потоковый шифр, который используется для шифрования и дешифрования данных.\hypertarget{index_usage}{}\doxysection{\texorpdfstring{Использование}{Использование}}\label{index_usage}
